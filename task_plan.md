# Task Plan
Questo file contiene il piano dei lavori per lo sviluppo dell'applicazione Wine Quiz con backend in Node.js e frontend in React.

## Fasi del progetto
1. Configurazione del Backend (Autenticazione & API)
2. Configurazione del Frontend (Pagine di Login e Struttura Base)
3. Elaborazione dei file (Estrazione dati e generazione Quiz su vino e additivi)
4. Sviluppo sistema di gioco (Quiz con sistema punteggi e leveling)
5. Quality Assurance (Review completa - Quality_Validator)
## Dettagli tecnici
- Entità principali: Utente (nome, cognome, email, password, livello, punteggio)
- Requisiti Sicurezza: Hash password
- Funzioni Core: Invio email simulato o con logger, controllo risposta quiz.
